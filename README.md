# ALU Project Work (Scilab)

## 📌 Project Overview

This project is a simple **Arithmetic Logic Unit (ALU)** simulation created using **Scilab**. It performs both arithmetic and logical operations based on user input.

The program allows the user to enter two numbers, choose an operation from the menu, and displays the result accordingly.

---

## 🚀 Features

### Arithmetic Operations
- Addition
- Subtraction
- Multiplication
- Division

### Logical / Bitwise Operations
- AND
- OR
- XOR
- NOT

---

## 🛠️ Technologies Used

- **Scilab**
- Input / Output Operations
- Conditional Statements
- Switch Case
- Bitwise Operations

---

## ▶️ How the Project Works

1. User enters two input numbers
2. Program displays operation menu
3. User selects the desired operation
4. ALU performs the selected operation
5. Final result is displayed on the screen

---

## 📘 Example Operations

### Arithmetic Example
Input:
- First Number: 10
- Second Number: 5
- Choice: Addition

Output:
- Result = 15

### Logical Example
Input:
- First Number: 6
- Second Number: 3
- Choice: AND

Output:
- Result = 2

## 🎯 Learning Outcomes

This project helps in understanding:

- Working of Arithmetic Logic Unit (ALU)
- Arithmetic and Logical Operations
- Switch Case Implementation
- User Input Handling
- Error Handling in Programming

---

## 📂 Project Purpose

The main purpose of this project is to simulate the basic functionality of an ALU used in computer systems and digital electronics.

It helps students understand how arithmetic and logic operations are performed inside a processor.
