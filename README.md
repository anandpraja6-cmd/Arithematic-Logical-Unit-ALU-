# Arithematic-Logical-Unit-ALU-
Implementation of Arithematic Logical Unit (ALU) using SCILAB 
 
📌 **Features**
The program supports the following operations:
➕ **Arithmetic Operations**
Addition
Subtraction
Multiplication
Division (with zero-check handling)
🔗 **Logical Operations**
AND
OR
XOR
NOT (on first number only)
🛠️ **Technologies Used**
Scilab (for simulation and implementation)
🚀 **How It Works**
User inputs two numbers.
A menu is displayed with operation choices.
User selects the desired operation.
The program performs the selected operation using a switch-case.
The result is displayed.
📂 **Code Structure**
clc; → Clears console
clear; → Clears variables
input() → Takes user input
switch-case → Controls operation flow
**Bitwise functions:**
bitand() → AND
bitor() → OR
bitxor() → XOR
bitcmp() → NOT
**Error Handling**
Prevents division by zero
Handles invalid menu choices
🎯 **Objective**
To understand the basic functioning of an ALU, including:
Arithmetic computations
Bitwise logical operations
Control flow using conditional statements
📚 **Learning Outcomes**
Basics of ALU design
Scilab programming
Bitwise operations
User input handling
